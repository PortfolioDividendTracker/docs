## Overview

The Portfolio Dividend Tracker API provides programmatic access to your portfolio data, transactions, dividends, and more through a RESTful interface.

Base URL: `https://staging.portfoliodividendtracker.com/api/v1`

## Authentication

The API uses **OAuth2 Authorization Code with PKCE** for authentication. All API endpoints require a valid Bearer token.

```json
"security": [
  {
    "oauth2": ["read", "write"]
  }
]
```

<Note>
  LLM Agents can fetch `/.well-known/oauth-authorization-server` to discover OAuth endpoints and dynamically register clients via the registration endpoint.
</Note>

## Rate Limiting

API requests are limited to **250 requests per minute** per authenticated user.
